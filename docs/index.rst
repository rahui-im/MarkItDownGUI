AIAgentDocument - MarkItDown GUI 문서
====================================

MarkItDown GUI는 다양한 문서 형식을 마크다운으로 변환하는 PyQt6 기반 데스크톱 애플리케이션입니다.

.. image:: https://img.shields.io/badge/Python-3.8+-blue.svg
   :target: https://python.org
   :alt: Python Version

.. image:: https://img.shields.io/badge/PyQt6-6.0+-green.svg
   :target: https://pypi.org/project/PyQt6/
   :alt: PyQt6 Version

.. image:: https://img.shields.io/badge/License-MIT-yellow.svg
   :target: https://opensource.org/licenses/MIT
   :alt: License

주요 기능
--------

* **다양한 형식 지원**: PDF, DOCX, PPTX, 이미지, 오디오 파일 등을 마크다운으로 변환
* **현대적인 UI**: PyQt6 기반의 반응형 사용자 인터페이스
* **다국어 지원**: 한국어, 영어 등 다국어 인터페이스
* **접근성**: WCAG 2.1 AA 준수 접근성 기능
* **테마 시스템**: 라이트/다크 모드 및 사용자 정의 테마
* **플러그인 아키텍처**: 확장 가능한 플러그인 시스템
